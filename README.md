# Databricks Data Optimization Lab

![Python](https://img.shields.io/badge/python-3.9+-blue.svg)

Lab de otimização de dados Delta no Databricks — OPTIMIZE, Liquid Clustering, VACUUM e Predictive Optimization com testes de sanidade.

## Descrição

Este repositório contém scripts e notebooks para demonstrar técnicas de otimização de dados no Databricks usando Delta Lake.

## Conteúdo

- `01_ingestao_dados.py` - Scripts de ingestão de dados
- `02_otimizacao.py` - Técnicas de otimização (OPTIMIZE, Liquid Clustering, VACUUM)
- `03_testes_sanidade.py` - Testes de validação e sanidade

## Tecnologias

- Databricks
- Delta Lake
- Apache Spark
- Python

## Como usar

1. Clone este repositório
2. Importe os scripts no seu workspace Databricks
3. Execute os notebooks na ordem numérica

## Recursos

- OPTIMIZE: Compactação de arquivos pequenos
- Liquid Clustering: Organização dinâmica de dados
- VACUUM: Limpeza de arquivos antigos
- Predictive Optimization: Otimização automática
